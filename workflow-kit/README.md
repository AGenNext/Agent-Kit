# Workflow Kit

Workflow Kit contains reusable workflow orchestration abstractions for AGenNext.

## Scope

```text
- workflow execution patterns
- multi-agent coordination
- dependency ordering
- shared state orchestration
- team execution patterns
- workflow adapters
```

## Migration source

The current implementation direction exists in:

```text
AGenNext/Agent-UI-Kit
  teams/
```

and related SDK workflow abstractions.

## Goal

Workflow orchestration should become a reusable platform kit instead of remaining embedded in a mixed-scope repository.
